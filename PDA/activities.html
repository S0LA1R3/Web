Minhas atividades

Atividades síncronas

Durante as aulas são feitas atividades para fixação de conteúdo. Neste momento um grupo aleátorio é criado e separado em salas para realização de uma atividade proposta para os próximos minutos.

Formulários de arremate

Ao término de cada aula é lançado um formulário de verificação de aprendizado, os quais temos 24h para realizar e são de extrema importância em nosso aprendizado.

Projeto Final Módulo 0

Uma carta para o futuro

No projeto final do módulo 0, Uma carta para o futuro, criamos um programa em scratch com personagens que interagiam entre si falando sobre seus objetivos e metas, cada personagem representava um objetivo e meta dos integrantes do grupo.
